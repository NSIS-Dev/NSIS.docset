ReadRegStr

Reads from the registry into the user variable $x. Valid values for rootkey are listed under WriteRegStr. The error flag will be set and $x will be set to an empty string ("") if the string is not present. If the value is present, but is of type REGDWORD, it will be read and converted to a string and the error flag will be set.

Parameters:

user_var(output) root_key sub_key name

Example:

ReadRegStr $0 HKLM Software\NSIS ""
DetailPrint "NSIS is installed at: $0"

History:

Added in NSIS v1.2g
