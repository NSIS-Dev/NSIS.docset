${LineSum}

Get sum of lines in text file.

Syntax

${LineSum} "[File]" $var

"[File]"      ; Input file
$var          ; Result: Sum of lines

Note:

  • Error flag if input file doesn't exist

Example

Section
    ${LineSum} "C:\a.log" $R0
    ; $R0="54"
SectionEnd
