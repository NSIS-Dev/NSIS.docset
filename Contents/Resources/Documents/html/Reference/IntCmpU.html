IntCmpU

Compares two unsigned integers val1 and val2. If val1 and val2 are equal, Goto jump_if_equal, otherwise if val1 < val2, Goto jump_if_val1_less, otherwise if val1 > val2, Goto jump_if_val1_more. Performs the comparison as unsigned integers.

Parameters

val1 val2 jump_if_equal [jump_if_val1_less] [jump_if_val1_more]

History

Added in NSIS v1.60
