LoadLanguageFile

Loads a language file for the construction of a language table. All of the language files that come with NSIS are in Contrib\Language Files After you have inserted the language file ${LANG_langfile} will be defined as the language id (for example, ${LANG_ENGLISH} will be defined as 1033). Use it with LangString, LicenseLangString, LangDLL and VIAddVersionKey.

Parameters

language_file.nlf

History

Added in NSIS v2.0 Alpha 3
