MiscButtonText

Replaces the default text strings for the four buttons (< Back, Next >, Cancel and Close). If parameters are omitted, the defaults are used. Accepts variables. If variables are used, they must be initialized in .onInit.

Parameters

[back_button_text [next_button_text] [cancel_button_text] [close_button_text]]
