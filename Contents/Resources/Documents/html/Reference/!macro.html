!macro

Creates a macro named 'macro_name'. All lines between the !macro and the !macroend will be saved. To insert the macro later on, use !insertmacro. !macro definitions can have one or more parameters defined. The parameters may be accessed the same way a !define would (e.g. ${PARAMNAME}) from inside the macro.

Parameters

`macro_name [parameter] [...]`

Example

!macro SomeMacro parm1 parm2 parm3
    DetailPrint "${parm1}"
    MessageBox MB_OK "${parm2}"
    File "${parm3}"
!macroend
