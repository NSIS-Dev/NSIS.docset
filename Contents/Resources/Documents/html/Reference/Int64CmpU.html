Int64CmpU

This function is only available when building a 64-bit installer.

Compares two unsigned integers val1 and val2. If val1 and val2 are equal, Goto jump_if_equal, otherwise if val1 < val2, Goto jump_if_val1_less, otherwise if val1 > val2, Goto jump_if_val1_more. Performs the comparison as unsigned integers. Same as IntCmpU, but treats the values as 64-bit integers.

Parameters

val1 val2 jump_if_equal [jump_if_val1_less] [jump_if_val1_more]

History

Added in NSIS v3.03
