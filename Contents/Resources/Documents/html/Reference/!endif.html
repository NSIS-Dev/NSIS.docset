!endif

This command closes a block started with !if, !ifdef, !ifndef, !ifmacrodef or !ifmacrondef.

Example

!ifdef VERSION
    OutFile installer-${VERSION}.exe
!else
    sOutFile installer.exe
!endif

!ifmacrodef MACRO
    DetailPrint "Macro defined" 
!else
    DetailPrint "Macro not defined" 
!endif
